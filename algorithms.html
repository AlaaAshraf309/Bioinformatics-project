Algorithms

Select any algorithm to analyze your DNA sequences. Each algorithm may require different inputs.

DNA Operations

GC Percentage / Reverse / Complement / Reverse Complement / Translation

O(n)

Perform basic DNA sequence operations such as GC content calculation, strand reversal, complement, reverse complement, and translation to amino acids.

Applications: Useful for sequence preprocessing and basic bioinformatics analysis.

Pattern Matching

Boyer-Moore (Bad Character & Good Suffix)

O(n + m)

Efficient pattern searching algorithm. Requires a pattern input and works on uploaded DNA sequences.

Applications: Detect motifs, primers, and sequence patterns.

Suffix Array

Suffix Array & Inverse

O(n log n)

Constructs suffix arrays and their inverse for string processing and fast searches.

Applications: Genome indexing, pattern search optimization.

Graph-Based Assembly

Overlap Graph

O(n^2)

Constructs overlap graphs to assist genome assembly. Upload sequences and optionally set minimum overlap length.

Applications: Genome assembly, contig merging.

Approximate Matching

Hamming & Edit Distance (Dynamic Programming)

O(n*m)

Find approximate matches between sequences using Hamming distance or edit distance. Requires pattern and sequence input.

Applications: SNP detection, approximate motif search, error-tolerant alignment.
